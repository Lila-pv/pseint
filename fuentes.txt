 Inicio Rapido===============Si solo quiere compilar y utilizar el programa instale las dependencias (ver requisitos) y utilice el comando "ming32-make win32" si esta en Microsoft Windows, o el comando"make linux3" si está en GNU/Linux o Mac Os. Los binarios se ubicarán en el directorio bin. Desde allí puede ejecutar "pseint" para comenzar a utilizar el programa. Requisitos para compilar PSeInt=================================Para compilar pseint se requieren las bibliotecas wxWidgets (versión 2.8 compilada en modo ansi,o 3.x compilada en modo unicode), y OpenGL. En  la mayoría de los sistemas GNU/Linux estas se pueden conseguir facilmente a través del repositorio de la distribución. Si utiliza Windows puede descargar el IDE ZinjaI que incluye el compilador gcc y las bibliotecas necesarias.  Acerca de los fuentes de PSeInt=================================Contenido de los directorios:  pseint: fuentes del interprete (verifica sintaxis e interpreta un algoritmo)  psdraw3: fuentes del editor de diagramas de flujo (permite editar el diagrama)  psdrawE: fuentes del exportador de diagramas de flujo (genera imagenes png/jpg/bmp)  psexport: fuentes del exportador (convierte de pseudocodigo a codigo C++)  wxPSeInt: fuentes de la nueva interfaz (editor de pseudocódigo, e interfaz principal)  psterm: fuentes de la terminal que se utiliza para ejecutar desde wxPSeInt  updatem: fuentes del programa que busca actualizaciones  test: pseudocódigos con casos de prueba y scripts para correrlos de forma automática  bin: demas archivos necesarios para ejecutar wxPSeInt (imagenes, documentacion, ayuda, etc).  dist: archivos adicionales para generar paquetes e instaladores  dtl: biblioteca auxiliar para comparar texto simil diff  hoedown: biblioteca auxiliar para convertir markdown a htmlPuede encontrar más información de la función de cada módulo y de cómo se comunican entreellos en http://cucarachasracing.blogspot.com.ar/2012/12/destripando-pseint.htmlAdentro de cada directorio hay varios Makefiles:  Makefile.lnx2 para compilar las versiones para GNU/Linux (usando wxWidgets 2.8.12 ansi)  Makefile.lnx3 para compilar las versiones para GNU/Linux (usando wxWidgets 3.1.x unicode)  Makefile.w32 para compilar las versiones para Windows desde Windows, con MinGW  Makefile.wine para compilar las versiones para Windows desde GNU/Linux, con Wine+MinGW  Makefile.m32 para compilar las versiones para Mac OS X i686 desde GNU/Linux, con IMCROSS  Makefile.ppc para compilar las versiones para Mac OS X powerpc desde GNU/Linux, con IMCROSS  Makefile.m64 para compilar las versiones para Mac OS X x64_64 desde GNU/Linux, con osxcrossPara compilar en o para Windows puede que necesite retocar las variables definidas en mingw.conf. Los archivos intermedios de la compilación se ubican en temp.Además, hay un archivo Makefile.pack que sirve para generar los paquetes e instaladoresde todo el software (solo desde GNU/Linux, y requiere wine+mingw+wxwidgets+nsis).Los fuentes son exactamente los mismos tanto para GNU/Linux como para Windows como para macOS.Todo está programado con C++03. El framework utilizado para el desarrollo de la GUI es wxWidgetsLos paquetes que se distribuyen desde el sitio oficial se construyen con gcc y en GNU/Linux.	 - linux-32 bits: en una máquina virtual con ubuntu 12.04, utlizando el script de 	                  dist/pseint-packer.sh	 - linux-64 bits: en dos máquinas virtuales, con ubuntu 12.04 (wx2+gtk2) y ubuntu 16.04					  (wx3+gtk3), utlizando el script de dist/pseint-packer.sh en cada una y					  combinando luego los resultados con el script dist/join.sh	 - windows: desde una GNU/Linux (slackware-current) utilizando mingw32 (la versión que se	            distribuye con ZinjaI) a través de wine, con Makefile.pack, y usando 	            nsis para construir el instalador	 - mac ppc/m32: desde una GNU/Linux (slackware-current) utilizando IMCROSS, con Makefile.pack	 - mac 64bits: desde una GNU/Linux (slackware-current) utilizando osxcross, con Makefile.packEn cada directorio de fuentes hay un archivo .zpr. Este es el archivo de proyecto para ZinjaI.Enlaces externos:	wxWidgets: http://www.wxwidgets.org	ZinjaI: http://zinjai.sourceforge.net	Hoedown: https://github.com/hoedown/hoedown.git    Diff Template Library (dtl): https://github.com/cubicdaiya/dtl.git	wine: http://winehq.org	NSIS: https://nsis.sourceforge.io/Main_Page	osxcross: https://github.com/tpoechtrager/osxcross	IMCROSS: http://www.sandroid.org/imcross/Actualizaciones: http://pseint.sourceforge.netPara enviar comentarios, consultas, reportes de errores, etc, acceda a los foros en:http://pseint.sourceforge.net/index.php?page=contacto.phpTodo se distribuye bajo licensia GPL (ver pack/help/gpl.html).Pablo Novara (zaskar_84@yahoo.com.ar)
